### Project import

You need the Android Studio v3.3.1 to open this project
  - clone the repo and checkout this PR branch
  - open the project on Android Studio
  - have fun (=
  - run unit tests with:
  ```sh
  $ ./gradlew lint test
  ```
  - run the instrumented tests with:  
  ```sh
  $ ./gradlew assembleDebugAndroidTest
  ```
